# Copy Indie Hackers Transcript

A Chrome extension that extracts Indie Hackers podcast transcripts and formats them as clean markdown (plus a plain-text export).

## What it does
- Pulls the full transcript from an Indie Hackers podcast episode page.
- Cleans up the text and formats it as readable markdown.
- Lets you copy the markdown or save a `transcript.txt` file.

## Example page
Use this episode to try it out:
`https://www.indiehackers.com/podcast/086-lynne-tye-of-key-values`

## How to use
1. Open an Indie Hackers podcast episode page in Chrome.
2. Click the extension icon.
3. Choose:
   - **Copy Markdown** to paste into notes or docs, or
   - **Save transcript.txt** to download a plain-text file.

## Output format
The markdown output is designed to be clean and scannable for notes and archives:
- Episode title
- Speakers and dialogue
- Minimal extra markup
